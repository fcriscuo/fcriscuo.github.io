Bio

I develop data analytics applications in support of bioscience research. My primary focus is on developing graph databases that integrate data from multiple scientific and clinical repositories. Current projects include a graph database that supports data related to ALS, a statistical analysis of synonymous mutations in cancer, and a graph database representation of the COSMIC database. My personal interests include cycling, music, and reading history. My favorite place to code is the Rose Main Reading Room in the NYC Public Library.
